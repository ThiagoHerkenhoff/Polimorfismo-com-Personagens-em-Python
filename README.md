# Polimorfismo-com-Personagens-em-Python
Projeto em Python demonstrando o conceito de Polimorfismo através de herança, utilizando diferentes classes de personagens com comportamentos sobrescritos.


Polimorfismo com Personagens em Python
📌 Sobre o Projeto

Este projeto demonstra o conceito de Polimorfismo dentro da Programação Orientada a Objetos (POO) utilizando Python.

A aplicação modela diferentes tipos de personagens (Guerreiro, Mago e Arqueiro), todos herdando de uma classe base comum (Personagens) e sobrescrevendo o mesmo método (falar()), mas com comportamentos diferentes.

O objetivo é demonstrar como múltiplos objetos podem responder ao mesmo método de formas distintas.

Objetivos Técnicos:

Aplicar herança entre classes

Demonstrar sobrescrita de métodos

Utilizar polimorfismo em tempo de execução

Trabalhar com listas de objetos

Escrever código organizado e orientado a objetos


Conceitos Aplicados:

Programação Orientada a Objetos
Herança
Polimorfismo
Sobrescrita de métodos
Lista de objetos
Iteração com for
Reutilização de código
